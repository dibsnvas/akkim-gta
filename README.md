# GTA: ALMATY — Akim Edition 🍎

A GTA 6–style third-person open-world walking game set in the streets of **Almaty, Kazakhstan**, where you play as the city's tireless mayor, **Akim Darkhan Satybaldy**. The comedy: he's an *obsessively* good mayor. No crime — only municipal excellence.

> *"Every pothole shall be filled. Every citizen shall be greeted."*

**A single self-contained HTML file.** Everything — geometry, textures, sound — is generated in code. The only external dependency is Three.js via CDN.

## ▶️ Play

- **Online:** open the [GitHub Pages site](https://armansu.github.io/gta-akim/) (enable Pages in repo Settings → Pages → Branch `main` if it isn't live yet).
- **Locally:** just open [`index.html`](index.html) in any modern browser.

## 🎮 Controls

| Action | Key |
| --- | --- |
| Walk | `WASD` / Arrow keys |
| Run | `Shift` |
| Look | Mouse (click to capture) |
| Interact / complete a deed | `E` |
| Wave to citizens | `Space` |
| Mute / unmute | `M` |
| Start | `Enter` (on title screen) |

On touch devices there's an on-screen joystick, a look-drag area, and `E` / 👋 buttons.

## 🏛️ The World

A recognizable, stylized Almaty:

- **Zailiysky Alatau** (Tian Shan) snowy peaks always on the southern skyline, in a warm GTA-6 pastel haze
- **Republic Square** with the Independence Monument and Kazakhstan flags
- **Panfilov Park** with the white-and-gold **Zenkov Cathedral**
- **Arbat** pedestrian street with cafes (КОФЕ, САМСА, ШАШЛЫК…)
- The glass **Esentai Tower** as the tallest landmark
- Apple trees everywhere (Almaty = "city of apples"), tree-lined avenues, yellow buses, taxis, traffic, fountains, benches, bus stops, crosswalks, and low-poly citizens

## 📋 Gameplay

Roam freely and complete **Akim Missions**: fix potholes, plant apple trees, repair bus stops, switch on fountains, clear litter & smog, and greet citizens (they cheer and pull out their phones for selfies). Each good deed bumps your **Approval Rating %** and adds tenge to the **City Budget**.

You can also hop into the black **"01 AKIM 777"** SUV and cruise the avenues — obeying the speed limit, of course.

## 🛠️ Tech

- [Three.js](https://threejs.org/) r128 (CDN) for rendering
- Procedural geometry, canvas-generated textures, and a WebAudio soundscape (traffic hum, cheers, chimes, horns)
- No build step, no other assets — one file
